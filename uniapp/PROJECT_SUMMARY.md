# UniApp 酒店无人超市移动应用 - 项目总结

## 项目概述

本项目成功实现了一个完整的酒店大堂无人售货超市移动应用，基于 uni-app 框架开发，可编译为微信小程序、支付宝小程序、H5、iOS/Android App 等多个平台。应用完整对接了后端 Spring Boot API，实现了商品浏览、购物车、订单管理、优惠券等核心业务功能。

## 技术亮点

### 2. 完整的业务功能

#### 用户系统
- **登录注册**: 完整的用户认证流程
- **个人信息**: 支持用户信息编辑
- **角色区分**: 支持管理员、员工、普通用户三种角色
- **员工专享**: 员工角色自动显示员工专享价格

#### 商品管理
- **首页展示**: 轮播图 + 商品推荐
- **分类浏览**: 左右分栏设计，便捷的分类导航
- **商品详情**: 完整的商品信息展示
- **扫码购物**: 支持条形码扫描快速查找商品
- **下拉刷新**: 实时更新商品数据
- **上拉加载**: 无限滚动加载更多商品

#### 购物车
- **智能管理**: 商品添加、删除、数量调整
- **全选功能**: 支持全选/单选商品
- **实时计算**: 自动计算商品总价
- **空车提示**: 优雅的空购物车提示页面

#### 订单系统
- **订单创建**: 从购物车一键创建订单
- **订单列表**: 查看所有历史订单
- **订单详情**: 完整的订单信息展示
- **订单取消**: 支持待支付订单取消
- **状态管理**: 待支付、已支付、已发货、已完成、已取消

#### 优惠券系统
- **优惠券列表**: 浏览所有可领取优惠券
- **一键领取**: 快速领取优惠券
- **我的优惠券**: 按状态分类管理优惠券
- **状态分类**: 可用、已使用、已过期
