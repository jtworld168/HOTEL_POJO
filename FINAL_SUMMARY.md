## 🎯 核心功能实现

### 1. 用户系统 🔐
- **三种角色**：管理员 (admin)、员工 (employee)、普通用户 (user)
- **认证方式**：JWT令牌认证 (SaToken)
- **密码加密**：MD5加密存储
- **用户信息**：包含头像URL，首页显示
- **默认账户**：admin / 123456

### 2. 商品管理 🛍️
- **商品信息**：名称、分类、价格、库存、图片、描述
- **唯一标识**：每个商品有唯一条形码
- **价格体系**：普通价格 + 员工内部价
- **库存管理**：自动扣减，防止超卖
- **商品详情页**：完整的商品信息展示

### 3. 员工内部价 💰
- **差异化定价**：员工享受特殊折扣价
- **自动识别**：员工登录后自动显示内部价
- **视觉区分**：员工价用绿色高亮显示
- **结算应用**：下单时自动使用员工价

### 4. 优惠券系统 🎫
- **券种类型**：折扣券、代金券
- **发放管理**：管理员创建和发放
- **用户领取**：一键领取，原子操作防超领
- **使用限制**：最低消费金额、有效期
- **订单抵扣**：下单时可选择使用优惠券

### 5. 购物车 🛒
- **添加商品**：选择商品加入购物车
- **数量调整**：增加、减少、删除
- **实时统计**：总价自动计算
- **徽章显示**：购物车图标显示商品数量
- **一键清空**：清空购物车功能

### 6. 订单系统 📦
- **创建订单**：从购物车生成订单
- **订单详情**：商品明细、价格、状态
- **订单列表**：查看历史订单
- **订单取消**：取消订单自动恢复库存
- **订单编号**：唯一订单号自动生成

### 7. 轮播图 🎠
- **首页展示**：3个轮播图
- **自动播放**：循环播放
- **点击跳转**：可跳转到商品详情
- **管理后台**：管理员可管理轮播图

### 8. 扫码功能 📷
- **调用摄像头**：使用ZXing库
- **识别条形码**：支持多种格式（EAN-13、CODE-128等）
- **自动跳转**：识别成功后跳转商品详情
- **权限处理**：浏览器摄像头权限请求

### 9. 国际化 🌍
- **完整翻译**：所有界面文字中英文双语
- **一键切换**：顶部导航栏语言切换按钮
- **持久化**：语言选择自动保存
- **默认语言**：中文

### 10. 美观UI 🎨
- **Element Plus**：专业UI组件库
- **响应式设计**：适配各种屏幕
- **简约风格**：现代、清爽、易用
- **流畅交互**：加载状态、过渡动画

## 🚀 快速开始

### 环境准备
- JDK 17+
- Maven 3.6+
- MySQL 8.0+
- Redis 6.0+
- Node.js 18+

### 3步启动

#### 1️⃣ 数据库初始化
```bash
mysql -u root -p < backend/src/main/resources/db.sql
redis-server
```

#### 2️⃣ 启动后端
```bash
cd backend
mvn spring-boot:run
```
✅ 启动成功后访问：http://localhost:8080/doc.html

#### 3️⃣ 启动前端
```bash
cd frontend
npm install
npm run dev
```
✅ 启动成功后访问：http://localhost:3000

## 📱 使用指南

### 登录系统
1. 访问 http://localhost:3000
2. 点击"登录"
3. 输入账号：`admin` 密码：`123456`
4. 登录成功，跳转到首页

### 浏览商品
1. 首页查看轮播图和商品列表
2. 点击商品卡片查看详情
3. 商品详情页显示条形码

### 购物流程
1. 商品详情页点击"加入购物车"
2. 点击顶部购物车图标（有徽章数字）
3. 调整商品数量
4. 点击"结算"创建订单
5. 查看"我的订单"

### 使用优惠券
1. 访问"优惠券中心"
2. 点击"立即领取"
3. 下单时选择使用优惠券

### 扫码功能
1. 点击顶部"扫码"按钮
2. 允许浏览器使用摄像头
3. 对准商品条形码扫描
4. 自动跳转到商品详情

### 切换语言
1. 点击顶部地球图标
2. 选择中文/English
3. 界面语言自动切换
